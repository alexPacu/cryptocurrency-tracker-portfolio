Data Dummy

Usage:
Fetch and Save ALL Coins: (USD)
npm run seed:all

Fetch with Custom Currency:
Fetch all coins with EUR prices:
node data_dummy/seedAllCoins.js EUR

Fetch all coins with GBP prices:
node data_dummy/seedAllCoins.js GBP
Clear existing data and fetch fresh:
node data_dummy/seedAllCoins.js USD --clear
